# Gemini / Antigravity Agent Instructions

Là Gemini / Antigravity agent đang làm việc trong project này, bạn **BẮT BUỘC** phải đọc và tuân thủ các tài liệu sau trước khi thực hiện bất kỳ hành động nào:

1. `working_rule.md` (ở root folder)
2. `.agents/README.md`
3. `.agents/rules/project_rules.md`
4. `.agents/TASK_STATUS.md` (BẮT BUỘC đọc trước khi làm task mới để nắm trạng thái)
5. Tất cả file trong `.agents/workflows/` bao gồm:
   - `1_understand_project.md`
   - `2_modify_code.md`
   - `3_final_review.md`
   - `4_update_context.md`

Hãy đảm bảo rằng bạn tuân thủ các workflows một cách cẩn trọng, thực hiện phân tích (Clarify First), đánh giá rủi ro (Data Leakage, Security) trước khi đưa ra bất kỳ cập nhật nào cho hệ thống, và cập nhật bối cảnh đầy đủ sau mỗi task.
